# DiamondPy
Small minecraft launcher for python

# Обозначения версий:

    - d -  в разработке
    - r - релиз
    - hf - хот фикс(считается актуальной)
    - b - бета(открытый бетатест, для него будет отдельная ветка)
пример: 0.1d (Версия 0.1 - в разработке)
